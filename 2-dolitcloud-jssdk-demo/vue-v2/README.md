# websdk-vue

>

## 运行步骤

``` text
1 进入src/components/HelloWorld.vue文件
var webServerAddr = ""; //渲染服务web地址，一般默认是渲染机ip+9000端口eg:http://10.0.0.60:9000
var connToken = "0";//从后台复制访问32位访问token（流路列表-查看地址 或者 云应用-点击最大流路数-查看地址）
2 修改以上两个变量为您云流渲染平台的实际值

3 运行服务
npm install
# serve with hot reload at localhost:8080
npm run dev

4 查看页面效果
```
